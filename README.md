System Overview
EquipBorrow allows us to efficiently manage the borrowing and returning of equipment, track availability, and maintain organized records of all transactions.

Features
Add, Edit, Delete, and View borrow records (CRUD)
Search records by borrower name, ID, or equipment
Filter records by status (Borrowed, Returned, Overdue)
Auto-detect overdue records
Dashboard stats (total, borrowed, returned, overdue)
Basic input validation
